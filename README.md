# qa_python
Тесты для BooksCollector

Как запустить: pytest -v tests.py

Что покрыто:
Добавление книг (add_new_book)
- можно добавить книгу с именем из одной буквы
- можно добавить книгу с именем ровно из 40 букв
- если имя длиннее 40 — не добавляется
- если добавить одну книгу дважды — дубля не будет
- у только что добавленной книги жанр пустой (это важно)

Получение всего словаря (get_books_genre)
- проверка, что возвращается правильный словарь вида {книга: жанр}

Установка жанра (set_book_genre)
- если книга есть и жанр допустимый — жанр ставится
- если жанр не из списка (например, «Роман») — книга остаётся без жанра
Список книг по жанру (get_books_with_specific_genre)
- например, запросили «Детективы» — получили только детективы, без фантастики

Детские книги (get_books_for_children)
- в детский список книг не попадают книги с возрастными жанрами: "Ужасы" и "Детективы"
- остальные жанры — попадают

Избранное (favorites)
-икнигу можно добавить в избранное, если она есть в словаре
- повторное добавление одной и той же книги не создаёт копию
- удаление из избранного работает
- если в избранном пусто, то метод возвращает пустой список
